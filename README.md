# Braille style sheets

A collection of regional CSS style sheets for braille formatting, for use in [eBraille](https://daisy.github.io/ebraille/#intro) and [DAISY Pipeline](https://daisy.org/activities/software/pipeline/), curated by [the DAISY Consortium](https://daisy.org/).

## The collection

The following style sheets are available. Follow the links for details:

- [United States and Canada (BANA)](bana)

## Contributing

If you have developed a style sheet for a region/language that you want to contribute, or if you propose a change to a style sheet already present in the repository, you may do so by heading over to our [development site on Github](https://github.com/daisy/braille-stylesheets) and by making a pull request. Alternatively, you may send an email to daisy-pipeline@mail.daisy.org.

## Issues

Issues in available style sheets may be reported through the [issue tracker](https://github.com/daisy/braille-stylesheets/issues). Please don't use the issue tracker to request new style sheets, as we don't have the capacity to develop a style sheet for each and every region. We rely on contributions from other organisations. For special requests you may contact us through email.
